  RENDER DEPLOYMENT - BACKEND INSTRUCTIONS
═══════════════════════════════════════════════════════════════

STEP 1: UPLOAD THIS FOLDER TO GITHUB
-------------------------------------
1. Go to github.com
2. Create new repository: "abrante3-deals-backend"
3. Upload ALL files from this folder
4. Commit

STEP 2: DEPLOY TO RENDER
-------------------------
1. Go to render.com
2. Sign up (free)
3. Click "New +" → "Web Service"
4. Connect GitHub
5. Select your "abrante3-deals-backend" repository
6. Configure:
   Name: abrante3-deals-api
   Runtime: Node
   Build Command: npm install
   Start Command: npm start
   Instance Type: Free

STEP 3: ADD ENVIRONMENT VARIABLES
----------------------------------
Click "Advanced" → Add these 5 variables:

1. PORT = 5000

2. MONGODB_URI = your_mongodb_connection_string
   (Get this from mongodb.com/cloud/atlas)
   Example: mongodb+srv://user:pass@cluster.mongodb.net/abrante3deals

3. JWT_SECRET = my_super_secret_key_at_least_32_characters_long

4. NODE_ENV = production

5. FRONTEND_URL = https://your-netlify-site.netlify.app
   (Your Netlify URL)

STEP 4: DEPLOY
--------------
Click "Create Web Service"
Wait 5-10 minutes

STEP 5: SEED DATABASE
---------------------
After deployment:
1. Click "Shell" tab in Render
2. Type: npm run seed
3. Press Enter
4. Wait for success message

Your backend URL will be:
https://abrante3-deals-api.onrender.com
